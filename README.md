# Pi-bot
### Roomba AI coding project

Developed a web-controlled Roomba using AI tools. 

Interfaced a Raspberry Pi with a Roomba, webcam, and LED.

Raspberry pi starts it's on web server for other devices (computers, phones) to drive & control LED brightness with a UI.
